# 🎓 Zakariya's Learning Hub - Getting Started

Welcome to your personalized educational website!

## Quick Start

### Option 1: Simple Local Access (Fastest)
1. Open the file: `educational-website/index.html`
2. Right-click on it and select **"Open with"** → **Your web browser**
3. The website will load immediately at `file:///C:/Users/saida/Downloads/my-extension/educational-website/index.html`

### Option 2: Using Python (if installed)
1. Open PowerShell in the `educational-website` folder
2. Run: `python -m http.server 8000`
3. Open your browser and go to: `http://localhost:8000`

### Option 3: Using Node.js (if installed)
1. Install http-server: `npm install -g http-server`
2. Navigate to the folder and run: `http-server`
3. Open the provided link in your browser (usually `http://localhost:8080`)

## How to Update Content

If you want to add, modify, or update any topic:
1. Open `index.html` in a text editor (VS Code recommended)
2. Find the relevant section (search for subject name)
3. Edit the content within the `<p>` tags
4. Save and refresh the browser
